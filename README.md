# TGIDN OneApp
Aplikasi kumpulan dokumen digital

## Cara Menjalankan

Buat aplikasi lalu jalankan pada perangkat mobile yang terhubung.
Pastikan perangkat sudah terhubung dengan benar dan mode debugging USB sudah diaktifkan.
```bash
flutter run
```

Buat file .APK untuk rilis
```bash
flutter build apk --release
```
